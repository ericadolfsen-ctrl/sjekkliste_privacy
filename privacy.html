Personvernerklæring – Sjekkliste_Y

Sist oppdatert: 11.12.2025

This privacy policy is currently written in Norwegian. If you have any questions in English, please contact support@sjekklistey.com.

1. Oversikt

Sjekkliste_Y lagrer alle sjekklistevalg, notater og bilder lokalt på enheten din. Ingen personopplysninger samles inn, lagres eksternt eller deles med tredjepart.

2. Hvilke data som håndteres

Appen samler ikke inn noen persondata til utvikler eller tredjepart. Følgende informasjon kan lagres lokalt på din enhet dersom du velger det:

  • Status for sjekkpunkter (OK / Ikke OK)
  • Frivillige notater du skriver inn
  • Bilder du tar i forbindelse med sjekklisten (opptil 3 per punkt)

Ingen av disse dataene forlater enheten uten at du selv aktivt deler PDF-rapporten via iOS sitt delingsark (for eksempel e-post eller meldinger).

3. Hvordan data brukes

Data brukes kun til:

  • å vise sjekklisten din i appen
  • å generere en PDF-rapport som du selv kan lagre eller dele

Appen har ingen sporing, annonser, analyseverktøy eller integrasjoner mot tredjepartstjenester.

4. Tredjepartstjenester

Appen bruker ingen tredjepartstjenester som samler inn personopplysninger.

5. Dine rettigheter og kontroll

Du kan når som helst slette all informasjon som appen har lagret lokalt ved å:

  • trykke «Ny sjekk – tøm liste» i appen, og/eller
  • slette appen fra enheten (alt innhold fjernes samtidig)
